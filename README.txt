The files inside of the 'JavaApplication' folder contain all of the source code for the JavaFX Application.
The files inside of the 'JavaApplicationExecutableFolder' contain everything needed for the Executable Jar File to run.
The files inside of the 'STM32Microcontroller' folder contain all of the source code for the STM32Mircocontroller, out of these, only the main.c has been edited to make Kinscope.
